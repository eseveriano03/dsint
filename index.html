Etiquetas de formulario

form: etiqueta padre de un formulario que permite enviar datos a un servidor local o externo.

input: capo de datos, campos de texto, caja de texto con las que el usuario puede interactuar.

button: representa un botón.

textarea: un campo de texto donde se puede introducir texto en líneas dependiendo de la longitud.

select: input de selección entre un conjunto de selecciones.

option: la opción que va dentro del select.
